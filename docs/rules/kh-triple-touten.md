# kh-triple-touten — 読点の過多（3個以上の連続）

- **ruleId**: `kh-triple-touten`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

読点「、」が3個以上連続している場合、文の構造を見直すことを推奨する。
3個以上の読点は文章の読みにくさや論理構成の問題を示すことが多い。
fix は提供しない（書き換えは文脈依存のため）。

## 出典

標準 校正必携 第8版（日本エディタースクール）組方原則および調整・句読点

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `春が来て、花が咲き、鳥が鳴いた。` | 指摘なし（2個） |
| 誤例 | `A、、、B。` | 3個連続を警告 |

## 偽陽性リスク

**中** — 口語体・列挙的文体では読点が多くなることがある。創作小説などでは状況に応じて無効化を検討。
