# kh-l3-iji-dokunn — 異字同訓の文脈依存使い分け

> ⚠️ ステータス: L3（LLM支援）・未実装。SDK が L3 を支持した後に実装予定。本ドキュメントは仕様の先行定義です。

- **ruleId**: `kh-l3-iji-dokunn`
- **nameJa**: 異字同訓の文脈依存使い分け
- **ガイドライン**: 標準 校正必携 第8版 (`kousei-hikkei-8`)
- **レベル**: L3（LLM支援）
- **想定 severity**: warning
- **適用モード案**: novel / official / academic

## 意図

同じ訓読みを持つ複数の漢字（異字同訓）を、文脈・語義に応じて正しく使い分けられているかを確認する。書籍は「使い分けとは異なる使い分けを否定する趣旨ではないが、一つの参考として提示する」と述べており、逸脱が明らかな場合に疑問として提起するルール。

## 書籍根拠

標準 校正必携 第8版（日本エディタースクール）異字同訓の漢字の使い分け例

- 前書き：使い分けに関わる年代差・個人差や各分野の表記習慣の違いがあることを踏まえ、ここに示す使い分け例は一つの参考として提示するものであると明記されている。
- 文化審議会国語分科会が 2014 年に報告した「異字同訓の漢字の使い分け例」を収録。常用漢字表に掲げられた複数の同訓字について、語義の説明と用例を対で示している。
- 133 項目（あ行〜わ行）にわたって各同訓字の語義と用例を掲示。「会う」と「遭う」、「明く」「明ける」「空く」「空ける」「開く」「開ける」、「上がる」「揚がる」「挙がる」など、文脈なしには判別できない対が多数収録されている。

## 動作原理

**LLM への入力**: 対象文書全体（または文・段落のブロック）と、本規則に対応する異字同訓ペアリスト（書籍収録の 133 項目）。

**LLM の判断基準**:
1. 文中の動詞・形容詞が書籍掲載の同訓字に該当するか識別する。
2. 書籍が示す語義（「会う」= 人と顔を合わせる、「遭う」= 好ましくない出来事に出くわす）と、文脈の意味が一致するかを判断する。
3. 不一致が明確な場合（例: 災難に「会う」）に警告を生成し、推奨字を提示する。
4. 書籍自体が「使い分けとは異なる表記を否定しない」と述べているため、あくまで疑問提起にとどめ、自動修正は行わない。

**LLM の出力**: 位置情報、検出された語、現在の字、推奨字、根拠（書籍語義の引用）の組。

## なぜ L1/L2 で不可能か

- 同訓字は発音が同一のため、正規表現では区別できない（L1 不可）。
- 形態素解析（L2）は品詞・活用形を分解できるが、「会う」と「遭う」の違いは同一品詞・同一活用形であり、文の意味的文脈（好ましい出来事か否か）でしか判別できない。
- 前後 1〜2 文の意味理解と語義の照合が必要であるため、L3（意味推論）が必須。

## 例

### Positive（指摘なし）

```
駅でばったり旧友に会った。
```
「会う」は「人と顔を合わせる」文脈で適切。指摘不要。

```
にわか雨に遭った。
```
「遭う」は「好ましくない出来事に出くわす」文脈で適切。指摘不要。

### Negative（指摘あり）

```
交通事故に会った。
```
「好ましくない出来事に出くわす」文脈で「会う」を使用。書籍の語義説明によれば「遭う」が適切。→ 警告: 「遭う（にわか雨に遭う・災難に遭う）」への変更を検討してください。

```
友人と駅で遭った。
```
「人と顔を合わせる」通常の対面文脈で「遭う」を使用。書籍の語義説明によれば「会う」が一般的。→ 警告: 「会う」への変更を検討してください。

## 偽陽性・運用注意

- 書籍自体が使い分け例は参考として提示するものであり異なる使い分けを否定しないと明記している。したがって severity は error でなく warning が上限。
- 文学作品やセリフでは意図的な字選択がある場合があり、著者の意図を尊重する必要がある。
- 使い分けの判断に「年代差・個人差」があることを LLM プロンプトに明示し、確信度が低い場合は指摘を抑制すること。
- 133 項目すべてをチェックすると指摘が多くなりすぎる恐れがある。実装時は誤用頻度が高いペア（遭う/会う、明ける/空ける/開ける 等）を優先的にカバーすることを推奨する。
