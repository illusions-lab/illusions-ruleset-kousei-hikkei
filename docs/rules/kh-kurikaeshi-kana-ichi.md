# kh-kurikaeshi-kana-ichi — かな一つ点「ゝ」の誤用

- **ruleId**: `kh-kurikaeshi-kana-ichi`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = info
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

かな一つ点「ゝ」（U+309D）・片仮名一つ点「ヽ」（U+30FD）は、現代文ではできるだけ使わないことが望ましい。
1950年「文部省刊行物表記の基準」で「々以外のくり返し符号はできるだけ使わないのが望ましい」と定められている。
本ルールはこれらの符号の出現を警告し、明示的な文字表記への書き換えを促す。

古典物・引用・復刻では原文尊重のためこのルールを無効化することを推奨する。

## 出典

標準 校正必携 第8版（日本エディタースクール）くり返し符号の使い方

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `かわいい子。` | 指摘なし |
| 誤例 | `ことゝ同じ。` | 「ゝ」を指摘（明示表記「とと」に書き換えを促す） |

## 偽陽性リスク

**低** — 「ゝ」「ヽ」自体の出現のみを検出。現代文では誤用がほとんど。
古典引用テキストでは false positive になりえるため、その場合はルールを無効化する。
