# kh-mixed-fw-hw-bracket — 全角括弧と半角括弧の混在

- **ruleId**: `kh-mixed-fw-hw-bracket`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

括弧の開きと閉じは同じ種類（全角同士または半角同士）で統一する。
「（」と「)」、「(」と「）」のような混在は誤りである。

## 出典

標準 校正必携 第8版（日本エディタースクール）組方原則および調整・括弧類

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `文章（補足）が続く。` | 指摘なし |
| 誤例 | `文章（補足)が続く。` | 全角閉じ「）」への修正提案 |

## 偽陽性リスク

**中** — 括弧内容が200文字を超えるケースは検出しない（パフォーマンスとバックトラック防止のため）。
入れ子括弧も対象外。ネストの深い括弧構造では手動確認を推奨。
