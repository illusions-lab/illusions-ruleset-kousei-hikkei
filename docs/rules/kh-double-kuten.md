# kh-double-kuten — 句点の重複（「。。」）

- **ruleId**: `kh-double-kuten`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

句点「。」が2個以上連続している場合は誤りである。句点は1個のみ使う。

## 出典

標準 校正必携 第8版（日本エディタースクール）組方原則および調整・句読点

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `終わった。次に進む。` | 指摘なし |
| 誤例 | `終わった。。次に進む。` | 重複句点を1個に修正提案 |

## 偽陽性リスク

**低** — 「。{2,}」の形のみを検出。単独句点は対象外。
