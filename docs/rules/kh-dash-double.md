# kh-dash-double — 全角ダーシは2個続けて使う

- **ruleId**: `kh-dash-double`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = info
- **適用モード**: 小説 / 公用文

## 意図

全角ダーシ「—」（U+2014）を文の区切りや言いさしに使う場合、単独では用いず、必ず「——」（2個）をひと組として使う。
半角ハイフン「-」・二分ダーシ「–」（U+2013）は対象外。

## 出典

標準 校正必携 第8版（日本エディタースクール）組方原則および調整・区切り約物

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `東京——そこが彼女の故郷だった。` | 指摘なし（2個） |
| 誤例 | `東京—そこが彼女の故郷だった。` | 1個を2個「——」に修正提案 |

## 偽陽性リスク

**低** — 偶数個は完全にスルー。半角ハイフンは対象外。
