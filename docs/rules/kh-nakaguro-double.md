# kh-nakaguro-double — 中黒「・」の重複

- **ruleId**: `kh-nakaguro-double`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

並列を示す中黒「・」（U+30FB）・半角中黒「･」（U+FF65）が2個以上連続するのは誤りである。
1個のみを使う。

## 出典

標準 校正必携 第8版（日本エディタースクール）組方原則および調整・中点類

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `東京・大阪・名古屋` | 指摘なし |
| 誤例 | `東京・・大阪` | 「・・」を「・」1個に修正提案 |

## 偽陽性リスク

**低** — 「[・･]{2,}」の形のみを検出。単独中黒は対象外。
