# kh-kurikaeshi-kanji-2 — 々は漢字1字の繰り返しにのみ使う

- **ruleId**: `kh-kurikaeshi-kanji-2`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

同の字点「々」（U+3005）は「漢字1字の繰り返し」にのみ用いる（人々・国々・年々など）。
漢字2字以上の語（民主主義・社会制度など）の直後に「々」を付けるのは誤用である。

## 出典

標準 校正必携 第8版（日本エディタースクール）くり返し符号の使い方

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `人々が集まった。国々の代表。` | 指摘なし |
| 誤例 | `民主主義々を唱える。` | 「々」（誤用）を指摘 |

## 偽陽性リスク

**低** — 漢字1字 + 々 は正しいとして除外。漢字2字以上 + 々 の形のみを検出。
