# kh-wave-dash-double — 波ダーシ「〜」の重複

- **ruleId**: `kh-wave-dash-double`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

範囲・程度を示す波ダーシ「〜」（U+301C）は1個で使う。
「〜〜」と2個以上続けるのは誤りである。

## 出典

標準 校正必携 第8版（日本エディタースクール）縦組の数詞表記の基準・数の幅を示す場合

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `東京〜大阪間を移動した。` | 指摘なし |
| 誤例 | `東京〜〜大阪間を移動した。` | 「〜〜」を「〜」1個に修正提案 |

## 偽陽性リスク

**低** — 「〜{2,}」の形のみを検出。単独「〜」は対象外。
