# kh-exclaim-kuten — 感嘆符・疑問符の直後に句点は不要

- **ruleId**: `kh-exclaim-kuten`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

感嘆符「！」・疑問符「？」（および半角 "!" "?"）は文末を表す機能を持つため、
直後に句点「。」を重ねる必要はない。「！。」「？。」は誤りである。

## 出典

標準 校正必携 第8版（日本エディタースクール）組方原則および調整・句読点

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `本当においしい！　またきます。` | 指摘なし |
| 誤例 | `本当においしい！。またきます。` | 句点「。」を削除提案 |

## 偽陽性リスク

**低** — 「[！!？?]。」の形のみを対象。感嘆符・疑問符の単独使用は対象外。
