# kh-leader-double — 三点リーダーは偶数個（2の倍数）で使う

- **ruleId**: `kh-leader-double`
- **ガイドライン**: 標準 校正必携 (`kousei-hikkei-8`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 小説 / 公用文 / ブログ / 学術 / SNS

## 意図

三点リーダー「…」（U+2026）は校正実務の慣行として必ず偶数個（2個、4個…）をひと組として使う。
1個・3個・5個など奇数個のみの使用は誤りとみなし、偶数個への修正を提案する。

## 出典

標準 校正必携 第8版（日本エディタースクール）組方原則および調整・区切り約物

## 例

| 区分 | テキスト | 期待 |
| --- | --- | --- |
| 正例 | `彼は黙って去っていった……。` | 指摘なし（2個 = 偶数） |
| 誤例 | `彼は黙って去っていった…。` | 1個を2個「……」に修正提案 |

## 偽陽性リスク

**低** — 偶数個の三点リーダーは完全にスルーする。奇数個は誤りが明確。
