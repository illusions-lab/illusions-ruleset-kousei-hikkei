# illusions-ruleset-kousei-hikkei

> illusions の校正ルールセット — **標準 校正必携 第8版**（日本エディタースクール）

日本の出版・校正の現場で長く「標準」とされてきた『校正必携』の約物（記号類）ルールを、
小説エディタ [illusions](https://illusions.app/) の自動校正として使えるようにしたものです。
原稿を書いている最中に、リーダーやダーシ、くり返し符号、句読点といった**記号まわりの
うっかりミス**をその場で指摘します。

## このルールセットでできること

文章を書くとき、記号の使い方は意外と崩れやすいものです。三点リーダーが1個だけになっていたり、
句点が二重になっていたり——こうした「校正者なら必ず直す」細かな約物の乱れを、illusions が
自動で見つけてくれます。プロの校正者が手元に置く規約集の知識を、執筆中のあなたが借りられる、
という位置づけです。

## 収録ルール

| ルール | 内容 | 例 |
| --- | --- | --- |
| 三点リーダーは偶数個で使う | 本文中の「…」は「……」（2個ひと組）が標準 | `去っていった…。` → `去っていった……。` |
| 全角ダーシは2個続ける | 区切り・言いさしの「—」は「——」（2個ひと組） | `東京—そこが故郷だった` → `東京——そこが故郷だった` |
| 「々」は漢字1字の繰り返しのみ | 2字以上の漢語の繰り返しに「々」は使えない | `民主主義々` → `民主主義` |
| かな一つ点「ゝ」「ヽ」 | 現代文では使用を避けることを推奨 | `ことゝ同じ` → `ことと同じ` |
| 感嘆符・疑問符の後に句点は不要 | 「！」「？」自体が文末を表す | `おいしい！。` → `おいしい！` |
| 句点の重複（。。） | 入力ミス・タイポの検出 | `終わった。。` → `終わった。` |
| 波ダーシの重複（〜〜） | 範囲・程度を示す「〜」は1個で使う | `東京〜〜大阪` → `東京〜大阪` |
| 中黒の重複（・・） | 並列の「・」は1個で使う | `東京・・大阪` → `東京・大阪` |

各ルールの詳細は [`docs/rules/`](./docs/rules/) を参照してください。

## 出典について — 『標準 校正必携』

『標準 校正必携』は、**日本エディタースクール出版部**が編集・刊行する校正実務の手引きで、
出版・印刷の現場で**事実上の標準**として広く使われてきたロングセラーです。書名の「必携」が
示すとおり、校正者が手元に常備する一冊として知られています。

- **約物（句読点・括弧・記号類）の名称と使い方**、くり返し符号、数字・単位の表記、
  JIS の校正記号など、原稿整理から校正までに必要な規約を簡潔にまとめています。
- 本ルールセットは、このうち**機械的に判定できる約物・記号のルール**を実装しています。
- 発行: 日本エディタースクール出版部 / 第8版（2011年）

> 校正の考え方を体系的に学びたい方には、原典の通読をおすすめします。
> 購入は [Amazon の検索結果](https://www.amazon.co.jp/s?k=%E6%A8%99%E6%BA%96%20%E6%A0%A1%E6%AD%A3%E5%BF%85%E6%90%BA%20%E7%AC%AC8%E7%89%88%20%E6%97%A5%E6%9C%AC%E3%82%A8%E3%83%87%E3%82%A3%E3%82%BF%E3%83%BC%E3%82%B9%E3%82%AF%E3%83%BC%E3%83%AB)から。

## illusions への追加

Electron 版 illusions で利用できます（Web 版は本体同梱ルールのみ）。

- **マーケットプレイス**: 本リポジトリには `illusions-ruleset` トピックが付いており、illusions が
  自動収集します。アプリの校正設定からルールセット一覧に表示され、有効化できます。
- **手動配置**: リリースの `dist/index.js` と `manifest.json` を
  `~/.illusions/rulesets/com.illusions-lab.kousei-hikkei/` に置きます。

有効化すると、「小説」「公用文」モードで該当ルールが自動的に働きます（一部のルールは全モードで有効）。
